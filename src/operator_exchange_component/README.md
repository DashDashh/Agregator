# Компонент внешнего обмена с эксплуатантами

Компонент содержит доверенный код, который обменивается сообщениями с внешними эксплуатантами дронов.

Kafka и MQTT-брокеры считаются внешней инфраструктурой и не входят в доверенный код компонента.
В доверенную часть входит только логика адаптеров, которая:

- публикует заявки эксплуатантам;
- публикует подтвержденную цену эксплуатанту;
- принимает сообщения `price_offer`;
- принимает сообщения `order_result`;
- обновляет состояние заказа через адаптер хранения.

Сейчас у компонента есть два backend-адаптера:

- `kafka`
- `mqtt`

Оба адаптера реализуют одинаковое поведение публикации, которое использует gateway publisher fan-out.
